# powerbi_mini_project
This project is a Power BI dashboard built using Sales, Products, and Area tables.

## Files Included
- powerbi_dashboard.pbix
- Source Excel file
- Dashboard screenshots

## Dashboard Features
- Budget Profit by State
- Budget COGS by Product
- Treemap by Product
- Slicer by Product Type
- Monthly and Quarterly analysis
- Synced slicers across pages

## Tools Used
- Power BI Desktop
- Excel
- Data Modeling
- Visualizations
